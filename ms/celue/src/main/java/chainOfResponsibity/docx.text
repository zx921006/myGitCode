责任链模式
    在论坛中发布文章
    后台要经过信息处理才能发表然后入库
实例：
    在项目中聊天的内容，房间标题等等,我们自定义一个接口，然后所有的xxFilter过滤方法都实现里面的doFilter,然后在过滤业务中创建一个
FilterChain(过滤链),然后把所有的过滤方法对象都加入到FilterChain链里面的List然后分别执行do方法，为了让多个责任链能够串联起来执行，
我们将FilterChain也实现Filter接口，我们就可以将第二个责任链加入到第一个责任链当中串行执行
    实现在一个链条上的一环返回false,则其他的过滤都不执行，方案是:在Filter接口设置一个返回值
